# 20_DEFINITION_OF_READY_DONE — GalFlowAI

## Definition of Ready

Uma história só entra em desenvolvimento se:

- [ ] Tem persona, ação e valor.
- [ ] É pequena o suficiente para uma sessão/sprint.
- [ ] Tem arquivo de contexto vinculado.
- [ ] Tem critérios de aceite em Gherkin.
- [ ] Tem plano de teste.
- [ ] Tem dependências/bloqueios mapeados.
- [ ] Não depende de suposição sem marcação.
- [ ] Foi priorizada no backlog.

## Definition of Done

Uma história só é concluída se:

- [ ] Critérios de aceite passaram.
- [ ] Testes foram criados/atualizados ou bloqueio justificado.
- [ ] Feature Preservation Matrix foi respeitada.
- [ ] Status Executivo foi atualizado.
- [ ] Daily Log foi atualizado.
- [ ] Gaps/TODOs foram atualizados.
- [ ] Commit pequeno foi criado.
- [ ] Relatório final foi entregue.
