# Stack técnica

## Base
- Windows
- Disco K:
- Python local via ambiente existente
- Gradio
- SQLite
- FFmpeg

## IA local
- WanGP/Wan2GP 1.3B como padrão de vídeo
- FramePack experimental
- Ollama para LLM local
- TTS offline como evolução

## Performance
- cenas curtas
- 480p/512p no preview
- 30 FPS no final via FFmpeg
- geração de vídeo uma cena por vez
- fila com lock de GPU
