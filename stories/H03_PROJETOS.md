# História H03 — Sistema de projetos

Como usuário, quero criar e retomar projetos para não perder contexto.

## Critérios de aceite
- Cria pasta de projeto com timestamp e slug.
- Cria subpastas brief, script, prompts, storyboard, renders, audio, final, logs.
- Salva project.json.
- Lista projetos existentes.
- Não sobrescreve projetos antigos.
