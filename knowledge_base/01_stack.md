# Stack do GalFlowAI

## MVP
- Python modular
- Gradio em 127.0.0.1:7860
- SQLite para metadados
- FFmpeg para montagem
- JSON por projeto
- logs em arquivo
- fallback de storyboard estático

- FramePack como motor experimental
- Ollama para LLM local quando disponível
- TTS offline como placeholder inicial
- Coqui/Kokoro como evolução posterior
