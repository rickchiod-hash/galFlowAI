# Design System

Nome visual: FlowForgeAI.

Estilo:
- escuro
- premium
- produtivo
- simples para usuário não técnico
- inspirado em ferramentas de criação com IA

Paleta:
- fundo principal: #0B1020
- cards: #111827
- cards secundários: #1F2937
- azul ação: #38BDF8
- verde sucesso: #22C55E
- amarelo alerta: #F59E0B
- vermelho erro: #EF4444
- texto principal: #F8FAFC
- texto secundário: #CBD5E1
- borda: #334155

Componentes:
- sidebar com etapas
- formulário de briefing
- cards por cena
- timeline horizontal
- upload de imagens
- galeria de renders
- console de logs
- botões de gerar roteiro, gerar cena, refazer cena, montar vídeo, abrir pasta final, cancelar
