# Ordem de uso no OpenCode

1. Abra o terminal.
2. Rode:

```bat
cd /d K:\AI_VIDEO_COMMERCIAL_STUDIO\opencodegalpasta
scripts\01_ABRIR_OPENCODE_AQUI.bat
```

3. Dentro do OpenCode, rode em ordem:

```text
/inicio
/analise
/fundacao
/ui
/projeto
/roteiro
/storyboard
/ffmpeg
/fila
/wangp
/tts
/testes
/docs
/qa
```

Observação: `/ffmpeg` vem antes de `/wangp` porque o sistema precisa funcionar sem IA pesada primeiro.
